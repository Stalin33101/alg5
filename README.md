# Лабораторная работа №5
# Бинарное дерево поиска (BST)

## Описание проекта
Реализация бинарного дерева поиска для хранения целых чисел с возможностью вставки, поиска, симметричного обхода и обхода в ширину (BFS).

## Функциональность

### Обязательная часть
- Вставка нового значения в дерево (`insert`)
- Поиск значения в дереве (`search`)
- Симметричный обход (inorder) — вывод значений в порядке возрастания

### Вариативная часть (пункт 10)
- **Обход в ширину (BFS)**: посещение узлов уровень за уровнем, слева направо

## Визуализация дерева

Вставляемые значения: `[50, 30, 70, 20, 40, 60, 80, 10, 35]`

### Структура дерева:
 ```
      50
     /  \
    30   70
   / \   / \
  20 40 60 80
 /    \
10    35
```
